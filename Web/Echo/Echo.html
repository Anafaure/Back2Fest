Histoire
Durant les festivals, les bouteilles en plastique sont souvent jetées ou laissées n'importe où. C'est pourquoi nous avons décidé de créer Echo, une bouteille réutilisable afin de réduire la pollution plastique. Echo est une bouteille connectée qui vous permet de commander vos boissons directement depuis votre smartphone. Elle est également personnalisable pour chaque festival.
Partenariats
Echo est en partenariat avec le festival Mélodica. Ce festival est un festival de musique électronique qui se déroule en plein air. Echo est un des partenaire officiel du festival, et espère continuer à se développer sur de nouveaux festivals.
Le produit
Echo est fabriquée à partir de matériaux recyclés et recyclables. Elle est également personnalisable pour chaque festival. Elle est équipée d'une puce NFC qui contient votre nom ainsi que des information sur vos commandes que vous pourrez sélectionner sur l'application en lien avec le festival partenaire. Elle est donc idéale pour les festivals et les évènements en plein air.
